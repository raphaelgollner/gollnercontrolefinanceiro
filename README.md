# 💰 Gollner Controle Financeiro

Uma aplicação web leve, rápida e responsiva para controle financeiro pessoal. Permite o registro de receitas e despesas, visualização de gráficos mensais e gerenciamento completo de fluxo de caixa, garantindo a privacidade dos dados de cada usuário.

---

## ✨ Funcionalidades

* **Autenticação Completa:** Sistema de login, cadastro e recuperação de senha.
* **Privacidade (RLS):** Cada usuário só tem acesso aos seus próprios lançamentos financeiros.
* **Gestão de Lançamentos:** Adição, edição, exclusão e visualização de receitas e despesas.
* **Categorização:** Fatiamento de gastos por categorias (Alimentação, Moradia, Transporte, Lazer, etc.).
* **Controle de Status:** Marcação rápida para lançamentos pagos ou pendentes.
* **Recorrência Anual:** Lançamentos marcados como recorrentes são replicados automaticamente para os meses subsequentes do ano ativo.
* **Resumo Visual:** Gráfico dinâmico em formato "Rosca" que mapeia as despesas mensais por categoria.
* **Exportação de Dados:** Geração automática de relatórios em `.csv` com os dados filtrados do mês.
* **Modo Escuro (Dark Mode):** Alternância fluida entre tema claro e escuro, com preferência salva no navegador.

---

## 🛠️ Tecnologias Utilizadas

* **Frontend:** HTML5, JavaScript Vanilla
* **Estilização:** Tailwind CSS (via CDN)
* **Gráficos:** Chart.js
* **Backend & Banco de Dados:** Supabase (PostgreSQL + Auth)
* **Hospedagem:** Netlify
